# hac - Home Assistant AI 助手

让 AI 帮你控制智能家居、创建自动化规则。通过自然语言对话，无需编写 YAML 配置。

## 快速开始

### 1. 安装

```bash
go build -o hac ./cmd/hac
```

### 2. 初始化

```bash
./hac init
```

按提示输入：
- Home Assistant 地址（如 `http://192.168.1.100:8123`）
- 长期访问令牌（在 HA 用户设置 → 安全 → 长期访问令牌 中创建）
- 配置仓库路径（如 `~/ha-config`，用于版本控制自动化配置）

### 3. 重启 Windsurf

### 4. 开始对话

在 Windsurf 中直接用自然语言：

**控制设备：**
- "帮我把客厅灯打开"
- "把卧室灯调到 50% 亮度"
- "关掉所有灯"

**创建自动化：**
- "帮我创建一个自动化：晚上7点到11点，客厅有人移动时自动开灯"
- "创建一个自动化：每天早上7点打开窗帘"
- "设置一个离家模式：我离开家时关闭所有灯和空调"

**管理自动化：**
- "列出所有自动化"
- "删除客厅感应灯自动化"
- "有哪些待确认的自动化？"

## 自动化创建流程

```
你: "创建一个自动化：晚上客厅有人时开灯"
    ↓
AI: 生成配置草稿，展示给你确认
    ↓
你: "确认" 或 "把亮度改成50%"
    ↓
AI: 部署到 Home Assistant，保存到配置仓库，提交 git
```

所有自动化配置都会保存到你指定的配置仓库，支持 git 版本控制。

## 命令行工具（可选）

除了 AI 对话，也可以直接使用命令行：

```bash
hac devices              # 列出所有设备
hac state light.客厅     # 查看设备状态
hac automations          # 列出所有自动化
hac sync                 # 同步 HA 配置到本地仓库
```

## 配置仓库结构

```
ha-config/
├── pending/           # 待确认的自动化草稿
├── automations/       # 已部署的自动化配置
├── scripts/
└── scenes/
```
